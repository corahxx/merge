## 🎯 项目背景与目标

### 项目背景

随着新能源汽车的快速发展，充电站建设需求日益增长。合理的选址决策直接影响充电站的运营效益和投资回报。传统的选址方法依赖人工经验和简单的地理位置判断，缺乏数据驱动的科学分析。

### 项目目标

基于现有的充电站数据资源和经纬度信息，开发一个智能化的充电站建站选址服务系统，为投资决策提供数据支撑和科学依据。

**核心目标：**
- 提供基于数据驱动的选址建议
- 分析现有充电站分布密度和竞争情况
- 识别高潜力建站区域
- 评估选址点的市场潜力和风险
- 提供可视化的选址分析报告

### 3. 核心算法设计

#### 3.1 距离计算算法

**Haversine公式（计算地球表面两点间距离）**
```python
from math import radians, cos, sin, asin, sqrt

def haversine(lon1, lat1, lon2, lat2):
    """
    计算两点间距离（公里）
    """
    # 转换为弧度
    lon1, lat1, lon2, lat2 = map(radians, [lon1, lat1, lon2, lat2])
    
    # Haversine公式
    dlon = lon2 - lon1
    dlat = lat2 - lat1
    a = sin(dlat/2)**2 + cos(lat1) * cos(lat2) * sin(dlon/2)**2
    c = 2 * asin(sqrt(a))
    r = 6371  # 地球平均半径（公里）
    
    return c * r
```

#### 3.2 竞争强度计算

```python
def calculate_competition_index(lon, lat, radius_km=5):
    """
    计算竞争指数
    """
    # 查询半径内的充电站
    nearby_stations = get_nearby_stations(lon, lat, radius_km)
    
    # 计算竞争指数
    competition_index = 0
    for station in nearby_stations:
        distance = haversine(lon, lat, station['longitude'], station['latitude'])
        # 距离越近，竞争越激烈（权重越大）
        weight = 1 / (distance + 0.1)  # 避免除零
        competition_index += weight * station['pile_count']
    
    return competition_index
```

#### 3.3 市场潜力评分模型

```python
def calculate_market_score(lon, lat, weights=None):
    """
    计算市场潜力评分（0-100分）
    """
    if weights is None:
        weights = {
            'population_density': 0.25,
            'traffic_flow': 0.20,
            'station_density': 0.20,  # 反向
            'poi_richness': 0.15,
            'ev_ownership': 0.10,
            'land_cost': 0.10  # 反向
        }
    
    # 获取各项指标
    population = get_population_density(lon, lat)
    traffic = get_traffic_flow(lon, lat)
    station_density = get_station_density(lon, lat, radius=5)
    poi_count = get_poi_count(lon, lat, radius=3)
    ev_ownership = get_ev_ownership(lon, lat)
    land_cost = get_land_cost(lon, lat)
    
    # 归一化到0-100
    scores = {
        'population_density': normalize(population, 0, 10000) * 100,
        'traffic_flow': normalize(traffic, 0, 100000) * 100,
        'station_density': (1 - normalize(station_density, 0, 20)) * 100,  # 反向
        'poi_richness': normalize(poi_count, 0, 100) * 100,
        'ev_ownership': normalize(ev_ownership, 0, 10000) * 100,
        'land_cost': (1 - normalize(land_cost, 0, 50000)) * 100  # 反向
    }
    
    # 加权平均
    market_score = sum(scores[key] * weights[key] for key in weights)
    
    return market_score
```

#### 3.4 密度聚类分析

```python
from sklearn.cluster import DBSCAN
import numpy as np

def identify_clusters(stations_data, eps_km=2, min_samples=3):
    """
    识别充电站聚集区域
    """
    # 提取经纬度
    coords = np.array([[s['longitude'], s['latitude']] for s in stations_data])
    
    # 转换为公里（近似）
    # 1度经度 ≈ 111km * cos(纬度)
    # 1度纬度 ≈ 111km
    lat_mean = np.mean(coords[:, 1])
    coords_km = coords.copy()
    coords_km[:, 0] *= 111 * np.cos(np.radians(lat_mean))
    coords_km[:, 1] *= 111
    
    # DBSCAN聚类
    clustering = DBSCAN(eps=eps_km, min_samples=min_samples, metric='euclidean')
    labels = clustering.fit_predict(coords_km)
    
    # 返回聚类结果
    return labels
```

```python
import requests

class AMapAPI:
    def __init__(self, api_key):
        self.api_key = api_key
        self.base_url = "https://restapi.amap.com/v3"
    
    def geocode(self, address):
        """
        地理编码：地址转经纬度
        """
        url = f"{self.base_url}/geocode/geo"
        params = {
            'key': self.api_key,
            'address': address
        }
        response = requests.get(url, params=params)
        return response.json()
    
    def regeocode(self, lon, lat):
        """
        逆地理编码：经纬度转地址
        """
        url = f"{self.base_url}/geocode/regeo"
        params = {
            'key': self.api_key,
            'location': f"{lon},{lat}"
        }
        response = requests.get(url, params=params)
        return response.json()
    
    def search_poi(self, lon, lat, radius=3000, types=''):
        """
        搜索周边POI
        """
        url = f"{self.base_url}/place/around"
        params = {
            'key': self.api_key,
            'location': f"{lon},{lat}",
            'radius': radius,
            'types': types,  # 如：060000（购物服务）、120000（商务住宅）
            'output': 'json'
        }
        response = requests.get(url, params=params)
        return response.json()
```
