## 🎯 问题背景

### 问题描述

在项目部署到不同计算机时，遇到MySQL端口被占用的问题。当前程序配置中**没有端口配置选项**，导致：

1. **无法灵活配置端口**：MySQL默认端口3306被占用时，无法使用其他端口
2. **部署不灵活**：不同环境（开发/测试/生产）可能需要不同端口
3. **配置不统一**：所有数据库连接都是硬编码，没有统一管理

### 业务影响

- ❌ **部署失败**：端口冲突导致无法连接数据库
- ❌ **环境隔离困难**：多环境部署时无法区分
- ❌ **维护成本高**：需要修改代码才能改变端口

## 🔧 技术可行性分析

### 1. 技术方案对比

#### 方案一：在config.py中添加port字段（推荐）

**优点：**
- ✅ 实现简单，修改量小
- ✅ 配置集中，易于管理
- ✅ 向后兼容（默认3306）
- ✅ 符合现有代码风格

**缺点：**
- ⚠️ 需要修改多个文件

**优点：**
- ✅ 代码复用性高

#### 2.1 创建数据库工具模块

**新建文件：`utils/db_utils.py`**

```python
# utils/db_utils.py - 数据库工具函数

from sqlalchemy import create_engine
from config import DB_CONFIG


def get_db_url() -> str:
    """
    获取数据库连接URL
    统一管理数据库连接字符串构建逻辑
    
    :return: 数据库连接URL
    """
    user = DB_CONFIG.get('user', 'root')
    password = DB_CONFIG.get('password', '')
    host = DB_CONFIG.get('host', 'localhost')
    port = DB_CONFIG.get('port', 3306)  # 默认3306
    database = DB_CONFIG.get('database', 'evcipadata')
    
    # 构建连接URL
    db_url = f"mysql+pymysql://{user}:{password}@{host}:{port}/{database}"
    
    return db_url


def create_db_engine(echo: bool = False):
    """
    创建数据库引擎
    统一管理数据库引擎创建逻辑
    
    :param echo: 是否打印SQL语句
    :return: SQLAlchemy引擎对象
    """
    db_url = get_db_url()
    return create_engine(db_url, echo=echo)
```

```python
# utils/db_utils.py - 数据库工具函数

from sqlalchemy import create_engine
from config import DB_CONFIG
import os


def get_db_url() -> str:
    """
    获取数据库连接URL
    支持环境变量覆盖配置
    
    :return: 数据库连接URL
    """
    # 优先使用环境变量，其次使用config.py配置
    user = os.getenv('DB_USER') or DB_CONFIG.get('user', 'root')
    password = os.getenv('DB_PASSWORD') or DB_CONFIG.get('password', '')
    host = os.getenv('DB_HOST') or DB_CONFIG.get('host', 'localhost')
    port = int(os.getenv('DB_PORT') or str(DB_CONFIG.get('port', 3306)))
    database = os.getenv('DB_NAME') or DB_CONFIG.get('database', 'evcipadata')
    
    # 构建连接URL
    db_url = f"mysql+pymysql://{user}:{password}@{host}:{port}/{database}"
    
    return db_url


def create_db_engine(echo: bool = False, **kwargs):
    """
    创建数据库引擎
    
    :param echo: 是否打印SQL语句
    :param kwargs: 其他create_engine参数
    :return: SQLAlchemy引擎对象
    """
    db_url = get_db_url()
    return create_engine(db_url, echo=echo, **kwargs)


def test_connection():
    """
    测试数据库连接
    
    :return: (success: bool, message: str)
    """
    try:
        engine = create_db_engine()
        with engine.connect() as conn:
            conn.execute("SELECT 1")
        return True, "数据库连接成功"
    except Exception as e:
        return False, f"数据库连接失败: {str(e)}"
```

### B. 数据库连接URL格式

**MySQL连接URL格式：**
```
mysql+pymysql://[user]:[password]@[host]:[port]/[database]
```

**示例：**
- 默认端口：`mysql+pymysql://root:password@localhost:3306/evcipadata`
- 自定义端口：`mysql+pymysql://root:password@localhost:3307/evcipadata`
- 省略端口（不推荐）：`mysql+pymysql://root:password@localhost/evcipadata`
