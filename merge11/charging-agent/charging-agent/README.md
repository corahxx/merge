# 充电桩数据管理系统

一个集成了智能问答、数据管理和账号管理的充电桩数据分析平台，支持大文件导入、数据清洗、智能分析和自然语言查询。

> **最后更新**: 2026-01-15  
> **版本**: 2.1（新增数据清洗功能）

---

## 📋 项目简介

本项目是一个面向充电桩行业的综合性数据管理平台，采用 **EVCIPA（中国充电联盟）** 风格UI设计。

### 核心模块

| 模块 | 功能 | 说明 |
|------|------|------|
| 🤖 **AI智能助手** | 自然语言查询数据库、智能对话 | 支持开关LLM功能 |
| 📊 **数据管理系统** | Excel导入、数据清洗、统计分析 | 仅管理员可导入 |
| 🧹 **数据清洗** | 重复检测、地址校验、软删除 | 管理员/操作员可用 |
| 🔍 **充电桩查询** | 按型号和省份查询充电桩 | 结果缓存5分钟 |
| 👥 **用户管理** | 账号管理、权限控制 | 管理员专属 |
| 📋 **审计日志** | 操作记录、登录日志 | 管理员专属 |

## 🚀 快速开始

### 方式一：EXE 启动器（推荐）

适合**非开发人员**或**新机器部署**：


| 默认账号 | 密码 |
|----------|------|
| `admin` | `Admin@2026` |

## ✨ 核心功能

### 🧹 数据清洗功能（v2.1 新增）

| 功能 | 说明 |
|------|------|
| **重复数据检测** | 完全重复（除UID外所有字段相同）、高度疑似（编号+地址+运营商+功率） |
| **地址一致性校验** | 对比省市区与场站地址，智能解析建议 |
| **软删除机制** | `is_active=0` 标记删除，支持恢复，不影响数据追溯 |
| **查询自动过滤** | 所有查询自动排除已删除数据 |

### 🔐 账号管理体系（v2.0 新增）

| 角色 | 权限 |
|------|------|
| 🔑 **管理员** (admin) | 全部功能 + 用户管理 + 数据导入 + 数据清洗 |
| 📝 **操作员** (operator) | 查询 + 统计 + 报告生成 + 数据清洗 |
| 👁️ **查看者** (viewer) | 仅查询和预览 |

- **登录认证**：bcrypt 密码加密
- **会话管理**：自动超时退出
- **审计日志**：记录所有关键操作

### 🤖 AI智能助手

- **自然语言查询**：用自然语言查询充电桩数据
- **智能意图识别**：自动识别用户意图（查询/对话/生成报告）
- **上下文理解**：支持跟进式对话
- **LLM开关**：可在启动器配置是否启用LLM功能
- **本地化部署**：基于 Ollama 本地大模型

### 📊 数据管理

- **大文件导入**：支持最大 600MB 的 Excel/CSV 文件
- **权限控制**：仅管理员可执行导入操作
- **随机预览**：数据预览随机抽取，避免偏差
- **多维度统计**：按运营商、区域、类型、功率统计
- **智能钻取**：区域饼图自动下钻到下一级
- **PDF报告导出**：生成专业的数据分析报告

## ⚙️ 配置说明

### 数据库配置

`config.py` 示例：

```python
DB_CONFIG = {
    'host': 'localhost',
    'port': 3306,
    'user': 'root',
    'password': 'your_password',
    'database': 'evcipadata'
}

# LLM配置（v2.0新增）
LLM_CONFIG = {
    'enabled': False,  # 是否启用LLM功能
    'model': 'qwen3:30b'
}
```

### 数据库表要求

系统需要以下数据库表：

| 表名 | 说明 |
|------|------|
| `evdata` | 充电桩数据主表 |
| `sys_users` | 用户账号表 |
| `sys_roles` | 角色表 |
| `sys_audit_logs` | 操作审计日志 |
| `sys_login_logs` | 登录日志 |

## 🐛 故障排除

| 问题 | 解决方案 |
|------|----------|
| **登录失败** | 检查用户名密码，默认 `admin` / `Admin@2026` |
| **数据库连接失败** | 检查 `config.py`，确认 MySQL 运行中 |
| **EXE闪退** | 检查是否复制了完整的目录结构 |
| **无法导入数据** | 确认使用管理员账号登录 |
| **LLM无响应** | 检查Ollama是否运行，或关闭LLM功能 |

📖 详细排查：**[docs/TROUBLESHOOTING.md](docs/TROUBLESHOOTING.md)**
