# 数据清洗重构可行性分析报告

**生成日期**: 2026-01-14  
**目标**: 重构数据清洗体系，实现98%以上数据准确率

---

## 一、项目背景

### 1.1 当前数据质量现状

基于数据质量检查（2026-01-14），发现以下问题：

| 问题类型 | 问题描述 | 影响记录数 | 占比 | 严重程度 |
|----------|----------|-----------|------|----------|
| 区县异常 | 天津市区县字段=城市名 | 73,256 | 100% | 🔴 严重 |
| 区县异常 | 重庆市区县字段=城市名 | 83,238 | 100% | 🔴 严重 |
| 编号缺失 | 充电站编号为空 | 456,522 | 11.1% | 🟡 中等 |
| 编号缺失 | 充电桩编号为空 | 382,044 | 9.3% | 🟡 中等 |
| 功率异常 | 额定功率=0 | 116,320 | 2.8% | 🟡 中等 |
| 功率异常 | 额定功率>1000kW | 2,017 | 0.05% | 🟠 低 |
| 数据重复 | 充电桩编号重复 | 431,859 | 10.5% | 🟡 中等 |
| 日期异常 | 生产日期=1970-01-01 | ~47万 | 11.5% | 🟡 中等 |

**总记录数**: 4,098,788 条

### 1.2 现有清洗逻辑不足

当前 `DataCleaner` 类的问题：
1. **清洗规则过于宽松**：只做基础的空白去除和去重
2. **缺少数据验证**：不验证数据逻辑合理性
3. **缺少异常记录**：问题数据直接丢弃或忽略，无法追溯
4. **缺少地址解析**：无法从地址中提取区县信息
5. **缺少编号生成**：空编号无法自动补全

## 二、重构目标

### 2.1 数据准确率目标

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 整体准确率 | ≥98% | 有效记录/总导入记录 |
| 关键字段完整率 | ≥99% | 运营商、省份、城市必填 |
| **运营商标准化率** | ≥95% | 运营商名称匹配标准字典 |
| 区县准确率 | ≥95% | 区县字段正确填充 |
| **区域隶属一致率** | ≥98% | 省-市-区县隶属关系正确 |
| 编号唯一性 | 100% | UID字段无重复 |
| **充电站ID有效率** | ≥95% | 充电站ID真实且一致 |
| 功率合理性 | ≥99% | 功率在0-1000kW范围内 |
| 经纬度有效率 | N/A | 源数据暂无，为空时跳过校验 |

**⚠️ 成功率阈值**：当单批次导入成功率低于 **95%** 时，系统将**自动暂停导入**，提示用户检查源数据质量。

### 2.2 功能目标

1. **严格数据验证**：每个字段有明确的验证规则
2. **智能数据修复**：自动修复可修复的问题
3. **异常数据记录**：所有异常数据记录到日志表
4. **数据血缘追溯**：每条记录可追溯来源文件和行号
5. **批量处理优化**：支持大文件高效处理

### 4.3 区县智能修复算法

```python
def fix_district(self, row):
    """
    从充电站位置中智能提取区县
    支持格式：
    - "北京市朝阳区xxx路xxx号" → 朝阳区
    - "天津市北辰区xxx" → 北辰区
    - "重庆市渝北区xxx" → 渝北区
    """
    address = row.get('充电站位置', '')
    province = row.get('省份_中文', '')
    
    # 直辖市列表
    direct_cities = ['北京市', '上海市', '天津市', '重庆市']
    
    if province in direct_cities and address:
        # 使用正则提取区县
        # 模式：城市名 + (区|县|市辖区)
        patterns = [
            r'(?:北京市|上海市|天津市|重庆市)([\u4e00-\u9fa5]{2,}(?:区|县))',
            r'([\u4e00-\u9fa5]{2,}(?:区|县))'
        ]
        
        for pattern in patterns:
            match = re.search(pattern, address)
            if match:
                district = match.group(1)
                # 验证是否是有效区县
                if self.is_valid_district(province, district):
                    return f"{province}{district}"
        
        # 无法提取，记录错误
        self.log_error(row, '区县_中文', 'EXTRACT_FAILED', 
                      f'无法从地址中提取区县: {address}')
    
    return row.get('区县_中文', '')
```

```python
def validate_operator(self, row):
    """
    严格校验运营商名称
    1. 检查是否为空
    2. 尝试别名匹配
    3. 模糊匹配标准运营商
    4. 记录异常
    """
    operator = str(row.get('运营商名称', '')).strip()
    
    if not operator:
        self.log_error(row, '运营商名称', 'REQUIRED_FIELD_MISSING',
                      '运营商名称为空')
        return '无登记运营商'  # 默认值
    
    # 1. 直接匹配标准名称
    if operator in STANDARD_OPERATORS:
        return operator
    
    # 2. 别名匹配
    if operator in OPERATOR_ALIASES:
        standard_name = OPERATOR_ALIASES[operator]
        self.log_error(row, '运营商名称', 'OPERATOR_ALIAS_CONVERTED',
                      f'运营商别名转换: {operator} → {standard_name}',
                      action='FIXED', fixed_value=standard_name)
        return standard_name
    
    # 3. 模糊匹配（包含关系）
    for std_operator in STANDARD_OPERATORS:
        if std_operator in operator or operator in std_operator:
            self.log_error(row, '运营商名称', 'OPERATOR_FUZZY_MATCHED',
                          f'运营商模糊匹配: {operator} → {std_operator}',
                          action='FIXED', fixed_value=std_operator)
            return std_operator
    
    # 4. 无法匹配，记录但保留原值
    self.log_error(row, '运营商名称', 'OPERATOR_NOT_IN_DICTIONARY',
                  f'未知运营商名称(不在标准字典中): {operator}',
                  error_level='WARNING')
    return operator  # 保留原值，但标记警告
```

### 4.5 区域隶属关系校验

**问题场景**：省份、城市、区县三者隶属关系不一致（如：省份=广东省，城市=北京市）

```python
def validate_region_hierarchy(self, row):
    """
    校验省份-城市-区县三级隶属关系
    如不一致，尝试从实际地址中提取修正
    """
    province = str(row.get('省份_中文', '')).strip()
    city = str(row.get('城市_中文', '')).strip()
    district = str(row.get('区县_中文', '')).strip()
    address = str(row.get('充电站位置', '')).strip()
    
    # 加载区域字典
    region_dict = self._load_region_dictionary()
    
    # 1. 校验省份是否有效
    if province and province not in region_dict['provinces']:
        # 尝试从地址中提取省份
        extracted_province = self._extract_province_from_address(address)
        if extracted_province:
            self.log_error(row, '省份_中文', 'PROVINCE_INVALID_FIXED',
                          f'省份无效已修正: {province} → {extracted_province}',
                          action='FIXED', fixed_value=extracted_province)
            province = extracted_province
        else:
            self.log_error(row, '省份_中文', 'PROVINCE_INVALID',
                          f'省份无效且无法从地址提取: {province}')
    
    # 2. 校验城市是否属于该省份
    if province and city:
        valid_cities = region_dict['province_cities'].get(province, [])
        if city not in valid_cities:
            # 尝试从地址中提取城市
            extracted_city = self._extract_city_from_address(address, province)
            if extracted_city:
                self.log_error(row, '城市_中文', 'CITY_NOT_IN_PROVINCE_FIXED',
                              f'城市不属于{province}已修正: {city} → {extracted_city}',
                              action='FIXED', fixed_value=extracted_city)
                city = extracted_city
            else:
                self.log_error(row, '城市_中文', 'CITY_NOT_IN_PROVINCE',
                              f'城市{city}不属于{province}，且无法从地址修正')
    
    # 3. 校验区县是否属于该城市（直辖市特殊处理）
    direct_cities = ['北京市', '上海市', '天津市', '重庆市']
    if province in direct_cities:
        # 直辖市：区县不应等于城市名
        if district == province or district == city:
            extracted_district = self._extract_district_from_address(address)
            if extracted_district:
                self.log_error(row, '区县_中文', 'DIRECT_CITY_DISTRICT_FIXED',
                              f'直辖市区县已修正: {district} → {extracted_district}',
                              action='FIXED', fixed_value=extracted_district)
                district = extracted_district
    else:
        # 非直辖市：校验区县隶属
        if province and city and district:
            valid_districts = region_dict['province_city_districts'].get((province, city), [])
            if district not in valid_districts:
                extracted_district = self._extract_district_from_address(address)
                if extracted_district:
                    self.log_error(row, '区县_中文', 'DISTRICT_NOT_IN_CITY_FIXED',
                                  f'区县不属于{city}已修正: {district} → {extracted_district}',
                                  action='FIXED', fixed_value=extracted_district)
                    district = extracted_district
    
    return province, city, district

def _extract_province_from_address(self, address: str) -> Optional[str]:
    """从地址中提取省份"""
    provinces = ['北京市', '上海市', '天津市', '重庆市', '广东省', '江苏省', 
                 '浙江省', '山东省', '河南省', '四川省', '湖北省', '湖南省',
                 '河北省', '福建省', '安徽省', '辽宁省', ...]  # 完整省份列表
    for prov in provinces:
        if prov in address or prov.replace('省', '') in address:
            return prov
    return None
```

### 4.6 充电站ID多重校验（保证真实性）

**问题场景**：充电站ID存在重复、不合理值，需要多重校验确保数据真实性

```python
def validate_station_id(self, row, existing_stations: dict):
    """
    充电站ID多重校验
    
    校验维度：
    1. 格式校验：非空、合理长度
    2. 重复检测：同一ID是否已存在
    3. 一致性校验：同一ID的运营商、地址是否一致
    4. 合理性校验：ID格式是否符合运营商命名规则
    """
    station_id = str(row.get('所属充电站编号', '')).strip()
    operator = str(row.get('运营商名称', '')).strip()
    address = str(row.get('充电站位置', '')).strip()
    station_name = str(row.get('充电站名称', '')).strip()
    
    # 1. 格式校验
    if not station_id or len(station_id) < 3:
        # 生成临时ID（基于运营商+地址哈希）
        temp_id = self._generate_temp_station_id(operator, address, station_name)
        self.log_error(row, '所属充电站编号', 'STATION_ID_EMPTY_GENERATED',
                      f'充电站编号为空，已生成临时ID: {temp_id}',
                      action='FIXED', fixed_value=temp_id)
        return temp_id, False  # False表示是生成的ID
    
    # 2. 重复检测 + 一致性校验
    if station_id in existing_stations:
        existing = existing_stations[station_id]
        
        # 检查运营商是否一致
        if existing['operator'] != operator:
            self.log_error(row, '所属充电站编号', 'STATION_ID_OPERATOR_MISMATCH',
                          f'站点ID {station_id} 运营商不一致: '
                          f'已有={existing["operator"]}, 当前={operator}',
                          error_level='WARNING')
        
        # 检查地址相似度
        address_similarity = self._calculate_address_similarity(
            existing['address'], address
        )
        if address_similarity < 0.5:  # 相似度低于50%
            self.log_error(row, '所属充电站编号', 'STATION_ID_ADDRESS_MISMATCH',
                          f'站点ID {station_id} 地址差异过大: '
                          f'相似度={address_similarity:.0%}',
                          error_level='WARNING')
            # 可能是不同站点误用了同一ID
            new_id = self._generate_unique_station_id(station_id, operator, address)
            self.log_error(row, '所属充电站编号', 'STATION_ID_REGENERATED',
                          f'检测到ID冲突，已生成新ID: {station_id} → {new_id}',
                          action='FIXED', fixed_value=new_id)
            return new_id, False
    else:
        # 新站点，记录到字典
        existing_stations[station_id] = {
            'operator': operator,
            'address': address,
            'station_name': station_name
        }
    
    return station_id, True  # True表示是原始ID

def _generate_temp_station_id(self, operator: str, address: str, name: str) -> str:
    """
    基于运营商、地址、站名生成临时站点ID
    格式: TEMP_{运营商缩写}_{地址哈希前8位}
    """
    import hashlib
    content = f"{operator}_{address}_{name}"
    hash_val = hashlib.md5(content.encode()).hexdigest()[:8].upper()
    operator_abbr = operator[:2] if operator else 'XX'
    return f"TEMP_{operator_abbr}_{hash_val}"

def _calculate_address_similarity(self, addr1: str, addr2: str) -> float:
    """
    计算两个地址的相似度（基于关键词重叠）
    """
    import re
    # 提取地址关键词
    keywords1 = set(re.findall(r'[\u4e00-\u9fa5]{2,}', addr1))
    keywords2 = set(re.findall(r'[\u4e00-\u9fa5]{2,}', addr2))
    
    if not keywords1 or not keywords2:
        return 0.0
    
    intersection = keywords1 & keywords2
    union = keywords1 | keywords2
    return len(intersection) / len(union)
```

### 4.8 经纬度验证修复算法

⚠️ **重要说明**：当前源数据中**暂无经纬度字段**，因此：
- 经纬度字段允许为空
- **为空时跳过所有校验**，不记录错误
- 仅当字段有值时才启动范围校验

```python
def fix_coordinates(self, row):
    """
    验证和修复经纬度坐标
    
    验证规则：
    - ⚠️ 为空时直接跳过，不校验
    - 经度范围：73.66°E ~ 135.05°E（中国境内）
    - 纬度范围：3.86°N ~ 53.55°N（中国境内）
    - 经纬度必须同时存在或同时为空
    """
    lng = row.get('经度')
    lat = row.get('纬度')
    
    # ⚠️ 关键：都为空时直接返回，不记录任何错误
    lng_empty = lng is None or lng == '' or (isinstance(lng, float) and pd.isna(lng))
    lat_empty = lat is None or lat == '' or (isinstance(lat, float) and pd.isna(lat))
    
    if lng_empty and lat_empty:
        return None, None  # 跳过校验，不记录错误
    
    # 只有一个为空，记录不完整
    if lng_empty != lat_empty:
        self.log_error(row, '经度/纬度', 'COORDINATE_INCOMPLETE',
                      f'经纬度不完整: 经度={lng}, 纬度={lat}')
        return None, None
    
    try:
        lng = float(lng)
        lat = float(lat)
    except (ValueError, TypeError):
        self.log_error(row, '经度/纬度', 'COORDINATE_INVALID_FORMAT',
                      f'经纬度格式无效: 经度={lng}, 纬度={lat}')
        return None, None
    
    # 范围验证
    if not (73.66 <= lng <= 135.05):
        self.log_error(row, '经度', 'LONGITUDE_OUT_OF_RANGE',
                      f'经度超出中国范围[73.66, 135.05]: {lng}')
        return None, None
    
    if not (3.86 <= lat <= 53.55):
        self.log_error(row, '纬度', 'LATITUDE_OUT_OF_RANGE',
                      f'纬度超出中国范围[3.86, 53.55]: {lat}')
        return None, None
    
    return round(lng, 6), round(lat, 6)
```

### 4.5 数据质量评分算法

```python
def get_quality_score(self, row):
    """
    计算单行数据质量评分 (0-100)
    
    评分维度：
    - 必填字段完整性: 40分
    - 数据格式正确性: 30分
    - 逻辑合理性: 20分
    - 可选字段完整性: 10分
    """
    score = 100
    issues = []
    
    # 1. 必填字段检查 (每缺一个扣10分，最多扣40分)
    required = ['运营商名称', '省份_中文', '城市_中文', '充电桩类型_转换']
    for field in required:
        if not row.get(field) or str(row.get(field)).strip() == '':
            score -= 10
            issues.append(f'{field}缺失')
    
    # 2. 格式检查 (每个问题扣5分，最多扣30分)
    if row.get('额定功率'):
        power = row.get('额定功率')
        if power <= 0 or power > 1000:
            score -= 5
            issues.append('功率异常')
    
    # 3. 逻辑检查 (每个问题扣5分，最多扣20分)
    province = row.get('省份_中文', '')
    district = row.get('区县_中文', '')
    if province in ['北京市','上海市','天津市','重庆市']:
        if district == province or not district:
            score -= 5
            issues.append('直辖市区县异常')
    
    # 4. 经纬度检查 (坐标不完整或无效扣5分)
    lng = row.get('经度')
    lat = row.get('纬度')
    lng_valid = lng is not None and lng != '' and 73.66 <= float(lng or 0) <= 135.05
    lat_valid = lat is not None and lat != '' and 3.86 <= float(lat or 0) <= 53.55
    if (lng_valid != lat_valid) or (lng_valid and not lat_valid):
        score -= 5
        issues.append('经纬度异常')
    
    # 5. 可选字段 (每缺一个扣2分，最多扣10分)
    optional = ['充电桩编号', '所属充电站编号', '充电站名称', '充电站位置', '额定功率', '经度', '纬度']
    missing_optional = 0
    for field in optional:
        if not row.get(field) or str(row.get(field)).strip() == '':
            missing_optional += 1
    score -= min(missing_optional * 2, 10)
    
    return max(0, score), issues
```
