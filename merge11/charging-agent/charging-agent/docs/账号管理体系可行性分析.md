## 二、技术方案

### 2.1 Streamlit 认证方案对比

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **streamlit-authenticator** | 成熟稳定，功能完整，支持密码哈希 | 需要额外配置文件 | ⭐⭐⭐⭐⭐ |
| **自建Session认证** | 完全可控，无额外依赖 | 需要自己实现安全机制 | ⭐⭐⭐ |
| **OAuth2/SSO集成** | 企业级，可对接现有系统 | 实现复杂，需要外部服务 | ⭐⭐ |
| **st.secrets + 简单密码** | 最简单快速 | 安全性低，不支持多用户 | ⭐ |

**推荐方案：streamlit-authenticator + 数据库存储**

#### 3.2.1 认证管理器 `auth/authenticator.py`

```python
# auth/authenticator.py

import streamlit as st
import bcrypt
from datetime import datetime
from typing import Optional, Dict
from services.user_service import UserService
from services.audit_service import AuditService

class Authenticator:
    """用户认证管理器"""
    
    def __init__(self):
        self.user_service = UserService()
        self.audit_service = AuditService()
    
    def login(self, username: str, password: str) -> Dict:
        """
        用户登录
        :return: {'success': bool, 'user': dict, 'message': str}
        """
        # 1. 查询用户
        user = self.user_service.get_user_by_username(username)
        if not user:
            self._log_login_failed(username, "用户不存在")
            return {'success': False, 'message': '用户名或密码错误'}
        
        # 2. 检查状态
        if user['status'] != 'active':
            self._log_login_failed(username, f"账户状态: {user['status']}")
            return {'success': False, 'message': '账户已被禁用或锁定'}
        
        # 3. 验证密码
        if not self._verify_password(password, user['password_hash']):
            self._log_login_failed(username, "密码错误")
            return {'success': False, 'message': '用户名或密码错误'}
        
        # 4. 登录成功
        self._log_login_success(username)
        self.user_service.update_last_login(username)
        
        # 5. 设置Session
        st.session_state['authenticated'] = True
        st.session_state['user'] = {
            'id': user['id'],
            'username': user['username'],
            'display_name': user['display_name'],
            'role': user['role'],
            'permissions': self._get_permissions(user['role'])
        }
        
        return {'success': True, 'user': st.session_state['user'], 'message': '登录成功'}
    
    def logout(self):
        """用户登出"""
        if 'user' in st.session_state:
            self.audit_service.log_action(
                username=st.session_state['user']['username'],
                action_type='logout',
                action_desc='用户登出',
                module='auth'
            )
        st.session_state['authenticated'] = False
        st.session_state['user'] = None
    
    def check_permission(self, permission: str) -> bool:
        """检查当前用户是否有指定权限"""
        if not st.session_state.get('authenticated'):
            return False
        user = st.session_state.get('user', {})
        return permission in user.get('permissions', [])
    
    def require_permission(self, permission: str):
        """权限装饰器：无权限时显示提示"""
        if not self.check_permission(permission):
            st.error("⛔ 您没有权限访问此功能")
            st.stop()
    
    def _verify_password(self, password: str, password_hash: str) -> bool:
        """验证密码"""
        return bcrypt.checkpw(
            password.encode('utf-8'), 
            password_hash.encode('utf-8')
        )
    
    def _get_permissions(self, role: str) -> list:
        """获取角色权限列表"""
        role_permissions = {
            'admin': ['user_manage', 'data_import', 'data_export', 'data_delete', 
                      'data_query', 'report_generate', 'system_config'],
            'operator': ['data_import', 'data_export', 'data_query', 'report_generate'],
            'viewer': ['data_query', 'report_generate']
        }
        return role_permissions.get(role, [])
```

#### 3.2.2 审计服务 `services/audit_service.py`

```python
# services/audit_service.py

import time
from datetime import datetime
from typing import Optional, Dict, Any
from sqlalchemy import text
from utils.db_utils import get_shared_engine

class AuditService:
    """操作审计服务"""
    
    def __init__(self):
        self.engine = get_shared_engine()
    
    def log_action(self,
                   username: str,
                   action_type: str,
                   action_desc: str,
                   module: str,
                   target_table: str = None,
                   target_id: str = None,
                   request_params: Dict = None,
                   result_status: str = 'success',
                   error_message: str = None,
                   affected_rows: int = 0,
                   execution_time_ms: int = None):
        """
        记录操作日志
        """
        try:
            with self.engine.connect() as conn:
                query = text("""
                    INSERT INTO sys_audit_logs 
                    (username, action_type, action_desc, module, target_table, 
                     target_id, request_params, result_status, error_message, 
                     affected_rows, execution_time_ms, created_at)
                    VALUES 
                    (:username, :action_type, :action_desc, :module, :target_table,
                     :target_id, :request_params, :result_status, :error_message,
                     :affected_rows, :execution_time_ms, NOW())
                """)
                conn.execute(query, {
                    'username': username,
                    'action_type': action_type,
                    'action_desc': action_desc,
                    'module': module,
                    'target_table': target_table,
                    'target_id': target_id,
                    'request_params': str(request_params) if request_params else None,
                    'result_status': result_status,
                    'error_message': error_message,
                    'affected_rows': affected_rows,
                    'execution_time_ms': execution_time_ms
                })
                conn.commit()
        except Exception as e:
            print(f"审计日志记录失败: {e}")
    
    def log_with_timing(self, func):
        """装饰器：自动记录执行时间"""
        def wrapper(*args, **kwargs):
            start = time.time()
            try:
                result = func(*args, **kwargs)
                execution_time = int((time.time() - start) * 1000)
                # 记录成功日志
                return result
            except Exception as e:
                execution_time = int((time.time() - start) * 1000)
                # 记录失败日志
                raise
        return wrapper
```

## 六、安全考虑

### 6.1 密码安全

- ✅ 使用 bcrypt 哈希存储密码（cost factor >= 12）
- ✅ 禁止明文传输和存储密码
- ✅ 密码强度要求：最少8位，包含字母和数字
- ✅ 登录失败次数限制（5次后锁定15分钟）

### 6.2 会话安全

- ✅ Session超时自动登出（默认30分钟）
- ✅ 单点登录（可选：同一账号只能一处登录）
- ✅ 登出后清除所有Session数据

### 6.3 审计安全

- ✅ 审计日志不可删除（只能由DBA清理）
- ✅ 记录敏感操作的完整参数
- ✅ 日志保留周期：建议90天以上
