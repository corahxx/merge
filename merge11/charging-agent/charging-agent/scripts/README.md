# 数据清洗工具脚本

本目录包含用于数据清洗和数据库维护的独立脚本。

## 脚本列表

| 脚本 | 功能 | 说明 |
|------|------|------|
| `run_name_sync.py` | 区域名称同步 | 将数据库中的省份/城市/区县名称与JSON字典同步 |
| `check_mysql.py` | MySQL连接检查 | 查看当前数据库连接状态和长时间运行的查询 |
| `kill_connections.py` | 终止阻塞连接 | 强制终止阻塞的数据库连接 |

## 使用方法

### 方式一：从项目根目录运行（推荐）

```bash
# 区域名称同步（自动确认模式）
python -u scripts/run_name_sync.py --yes

# 区域名称同步（手动确认）
python scripts/run_name_sync.py

# 检查MySQL连接
python scripts/check_mysql.py

# 终止阻塞连接
python scripts/kill_connections.py
```

### 方式二：进入scripts目录运行

```bash
cd scripts

# 区域名称同步
python -u run_name_sync.py --yes

# 检查MySQL连接
python check_mysql.py

# 终止阻塞连接
python kill_connections.py
```

## 脚本详情

### run_name_sync.py

**功能：** 将数据库中的区域名称（省份_中文、城市_中文、区县_中文）与 `data/region_code_mapping.json` 字典同步。

**参数：**
- `--yes` 或 `-y`：自动确认执行，不需要手动输入

**输出示例：**
```
============================================================
区域名称同步脚本 (PyMySQL直连)
============================================================

[1/4] 加载JSON映射...
  - 省份: 34
  - 城市: 384
  - 区县: 3395

[2/4] 同步省份名称...
    [130000] 河北省: +100 条
    [310000] 上海市: +241 条
  省份完成: 345 条

[3/4] 同步城市名称...
  城市完成: 12,345 条

[4/4] 同步区县名称...
  区县完成: 5,678 条

============================================================
同步完成!
  - 省份修复: 345 条
  - 城市修复: 12,345 条
  - 区县修复: 5,678 条
  - 总计: 18,368 条
  - 耗时: 320.5 秒
============================================================
```

### check_mysql.py

**功能：** 查看MySQL当前连接状态，识别长时间运行的查询，可选择终止。

**输出示例：**
```
=== 当前MySQL进程 ===
ID: 123, User: root, DB: evcipadata, Time: 5s, State: Query, Info: SELECT...

总连接数: 10

=== 长时间运行的查询 (>10秒) ===
  ID: 456, Time: 120s, Command: Query, Info: UPDATE evdata...

发现 1 个长时间运行的查询
是否终止这些进程? (y/n):
```

### kill_connections.py

**功能：** 强制终止所有evcipadata数据库的连接（除了当前连接和系统进程）。

**使用场景：** 当数据库出现锁等待超时或连接阻塞时使用。
