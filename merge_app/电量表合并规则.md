# 电量表多表合并规则

本文档描述「电量表多表合并」模块的 Sheet 选择、表头判定、仅合并与合并汇总的完整规则。**通过后再执行开发。**

示例表路径：`C:\Users\HONOR\Desktop\充电数据示例\电量`。UI 与「公共桩多表合并」「充电站多表合并」类似：左侧新增功能入口，导入表格后提供两个按钮（见下）。

---

## 一、总则

- **支持格式**：Excel（.xlsx / .xls）、CSV（与现有合并模块一致）。
- **合并策略**：多文件上传后优先合并能合并的，不能合并的文件单独列出并写明原因。
- **功能入口**：左侧新增「**电量表多表合并**」，与公共桩、充电站、数据清洗并列；主区 UI 与另外两个合并模块类似（标题区、文件上传、结果区）。
- **导入表格后**：提供两个按钮——「**仅合并**」与「**合并汇总**」（见第三节、第四节）。

---

## 二、Sheet 选择规则

1. **有内容的 Sheet**：指该 Sheet 前若干行非空单元格数达到一定数量（如 ≥ 3），视为有内容；无内容 Sheet 不参与选择。
2. **仅一个有内容 Sheet**：直接以该 Sheet 作为数据源，不再判断命名。
3. **多个有内容 Sheet**：
   - **若所有有内容 Sheet 的名称均为「sheet1」「sheet2」「sheet3」等形式**（即按 sheet1、sheet2 这种形式命名）：则选用名称为 **「sheet1」** 的 Sheet 作为数据源进行合并（仅用 sheet1 一张表）。
   - **若 Sheet 按日期命名**（如含 2024-01、2024-02 或可解析的日期）：则选用**日期最新**的那个 Sheet 作为数据源。
   - 若既非 sheet1/sheet2 形式又非日期命名，则需约定默认规则（如取第一个有内容的 Sheet），可在实现时补充。

---

## 三、表头判定规则

- 在选定的 Sheet（或单 Sheet 文件）中，查看**前 3 行**。
- **表头行**：前 3 行中**最先出现**以下任一关键词的那一行，视为表头行：
  - **【省级行政区域名称】**
  - **【月度充电电量】**
- 表头行之下均为**数据行**（合并内容或汇总来源）。
- 若 3 行内均未出现上述两个关键词，则报错：该文件/Sheet 无法确定表头，列入未合并列表并写明原因。

## 四、仅合并功能

- 确定表头后，先对每个文件的数据**新增以下内容**，再进行**纵向拼接**（多文件则多表纵向拼成一张表）。
- **新增两列**（置于合适位置，如最前或「上报机构」位置）：
  1. **文件名称**：用**表格（文件）的名字**填充该文件对应的所有行。
  2. **运营商名称**：根据**文件名称**按下方映射规则清洗得到；无匹配时填「**未识别**」。
- **新增七个字段**（取值方式与第五节 5.2 相同）：
  - **直流桩电量**、**交流桩电量**、**私家车使用电量**、**公交车使用电量**、**出租车使用电量**、**环卫物流车使用电量**、**其他使用电量**

## 五、合并汇总功能

### 5.1 新增字段（仅合并与合并汇总共用）

**仅合并**与**合并汇总**均先按 5.2 规则新增以下七个字段（若原表无对应列则新建，若有则按规则覆盖或计算）。仅合并在新增后**纵向拼接**各表；合并汇总在新增后再按 5.3 **按省级行政区域加总**输出。

- **直流桩电量**
- **交流桩电量**
- **私家车使用电量**
- **公交车使用电量**
- **出租车使用电量**
- **环卫物流车使用电量**
- **其他使用电量**

### 5.2 新增字段取值规则

- **数值清洗（先行）**：以下字段在参与计算或汇总前，须先做数值清洗，保证为数值后再使用；**最终导出的表中这些字段也以清洗后的数值输出**（无 KWh、KWH 等单位，无 -、/ 等空值字符）。涉及字段：**月度充电电量**（含列名如「月度充电电量（KWh）」）、**直流桩**、**交流桩**、**私家车**、**公交车**、**出租车**、**环卫物流车**、**其他**。
  - **去掉单位**：列值中的单位字样均去掉，如 **KWh**、**KWH**、**（KWh）** 等（不区分大小写），只保留数字。
  - **去掉代表空值的字符**：将表示空值的字符删掉并视为空，如 **-**、**/**；若整格仅为该类字符或为空，则视为空值（后续按空值处理，导出时以 0 或空体现）。
- 先根据**原表**中与「直流桩」「交流桩」「私家车」「公交车」「出租车」「环卫物流车」「其他」对应的列（列名需包含这些关键词或与上述新增字段一一对应）取值。
- **判定**：若「直流桩」列下面的值**有百分数**（如 30%、0.5、或者是存在一个及以上%）或**全部数值型单元格的值 ≤ 100**，则视为比例；否则视为绝对值。
- **若为比例**（有百分数或全部数值型 ≤ 100）：
  - 直流桩电量 = **月度充电电量 × 直流桩**（对应列）
  - 交流桩电量 = **月度充电电量 × 交流桩**
  - 私家车使用电量 = **月度充电电量 × 私家车**
  - 公交车使用电量 = **月度充电电量 × 公交车**
  - 出租车使用电量 = **月度充电电量 × 出租车**
  - 环卫物流车使用电量 = **月度充电电量 × 环卫物流车**
  - 其他使用电量 = **月度充电电量 × 其他**
  - （若为百分数，需先转换为小数再乘，如 30% → 0.3。）
- **若为绝对值**（不全是百分数且不全是 ≤ 100）：
  - 直流桩电量 = 直流桩（列值）
  - 交流桩电量 = 交流桩（列值）
  - 私家车使用电量 = 私家车（列值）
  - 公交车使用电量 = 公交车（列值）
  - 出租车使用电量 = 出租车（列值）
  - 环卫物流车使用电量 = 环卫物流车（列值）
  - 其他使用电量 = 其他（列值）
- **空值处理**：上述七个字段（直流桩电量、交流桩电量、私家车使用电量、公交车使用电量、出租车使用电量、环卫物流车使用电量、其他使用电量）若为空值，则填充为 **0**。

## 六、错误与未合并

- 单文件无法确定表头、Sheet 选择失败、读表异常等，均列入未合并列表，并写明原因（如「无法确定表头」「未找到 sheet1」等）。
- 未合并文件以表格展示「文件名」「未合并原因」，与现有合并模块一致。

## 七、实施顺序建议

1. 确认本规则文档无歧义或补充需求后，再开发。
2. 开发顺序建议：左侧新增「电量表多表合并」入口 → 实现 Sheet 选择与表头判定 → 实现「仅合并」→ 实现「合并汇总」→ 与现有两合并模块 UI 对齐（大/小文件、未合并表格等可按需简化或保持一致）。
